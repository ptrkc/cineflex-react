## How to run

1. Clone this repository
2. Copy the file `.env.example` to your own `.env`. This app uses a provided API.

```bash
cp .env.example .env
```

3. Install the dependencies

```bash
npm i
```

4. Run it with

```bash
npm start
```

5. Open http://localhost:3000 on your browser
